# File XOR Modifier

GUI-приложение на C++ и Qt для поточной модификации файлов с использованием логической операции XOR (с 8-байтным ключом).

##  Особенности реализации
* Многопоточность: Тяжелые операции чтения/записи вынесены в отдельный рабочий поток (`QThread` + `Worker`), благодаря чему графический интерфейс (UI) остается полностью отзывчивым и не "замирает" даже при обработке гигабайтных файлов.
* Экономия памяти: Файлы читаются и обрабатываются буферами по 1 МБ, что позволяет безопасно модифицировать файлы огромного размера без переполнения ОЗУ.
* Отслеживание прогресса: В реальном времени обновляется QProgressBar и статус-бар.
* Безопасная остановка: Использование QAtomicInt для безопасного прерывания потока по кнопке "Стоп".

##  Функционал
- [x] Настройка маски входных файлов (например, `*.txt *.bin`).
- [x] Опциональное удаление входных файлов после обработки.
- [x] Выбор пути для сохранения результирующих файлов.
- [x] Обработка конфликтов имен: перезапись или модификация имени (добавление счетчика).

- [x] Выполнение операции XOR с 8-байтной переменной (16 HEX символов), введенной с формы.
